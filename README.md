繰り返してる あの日を
かける言葉はもう、冷えきって
仕舞い込んだから、きっと大丈夫
覚めて、足りない、
渇ききってひび割れてもう、涙もない
さだめ、おまえとわたしだけ
痛み、迷いも捨てゆけ

しわくちゃになった答えを
わたしを越え、

定めと言うだろう
さだめ、おまえとわたしだけ
甘い夢など、捨てゆけ
行方知れずの、静寂から
さだめ、おまえを刻むよ

利用し合う世界も
